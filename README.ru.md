Легковесный музыкальный проигрыватель для Windows с расширенными функциями и интуитивным интерфейсом.

## Возможности

- **Компактный размер**: Всего ~1MB
- **Поддерживаемые форматы**: MP3, WAV, FLAC, AAC, OGG, WMA, M4A
- **Расширенное управление воспроизведением**:
  - Воспроизведение/Пауза/Следующий/Предыдущий
  - Прогресс-бар с перемоткой по клику
  - Регулировка громкости с функцией отключения звука
  - Навигация по истории прослушивания
- **Режимы воспроизведения**:
  - По порядку (последовательно)
  - Случайный (перемешанный)
  - Повтор (одного трека)
  - Повтор всего (всего списка)
- **Умное управление библиотекой**:
  - Сканирование папок с отслеживанием прогресса
  - Поиск в реальном времени с подсветкой
  - Система избранных треков с сохранением настроек
  - Сохранение/загрузка плейлистов (формат M3U)
- **Подробная информация о треках**:
  - Отображение длительности (чч:мм:сс)
  - Битрейт и размер файла
  - Полные метаданные (исполнитель, альбом, жанр, год)
  - Отображение обложки альбома
  - Дата создания файла
- **Пользовательский опыт**:
  - Поддержка перетаскивания файлов и папок
  - Горячие клавиши для всех основных функций
  - Визуальные индикаторы избранных треков
  - Пользовательский рендеринг списка
  - Сохранение настроек

## Системные требования

- **ОС**: Windows 7 или новее
- **.NET Framework**: 4.7.2 или новее
- **Место на диске**: Менее 2MB
- **ОЗУ**: Минимум 512MB

## Установка

1. Скачайте последнюю версию с GitHub
2. Распакуйте ZIP-файл в предпочитаемую папку
3. Запустите `MusicLocalUI.exe`

### Необходимые файлы:
- `MusicLocalUI.exe` (Основное приложение)
- `TagLibSharp.dll` (Библиотека для метаданных)

## Как использовать

### Базовое воспроизведение:
1. Нажмите "Open" для выбора папки с музыкой
2. Нажмите "Scan" для загрузки всех аудиотреков
3. Двойной клик по любому треку для начала воспроизведения
4. Используйте элементы управления: ⏸ ▶ ⏭ ⏮

### Расширенные функции:
- **Поиск**: Вводите текст в поисковую строку для фильтрации треков
- **Избранное**: Нажмите "♡ Favorite" для отметки любимых треков
- **Плейлисты**: Используйте "Save Playlist"/"Load Playlist" для управления сессиями
- **История**: Используйте кнопки Назад/Вперед для навигации по истории
- **Режимы просмотра**: Переключайтесь между "Show All" и "Show Favorites"

### Горячие клавиши:
- `Пробел` - Воспроизведение/Пауза
- `Ctrl+Вправо` - Следующий трек
- `Ctrl+Влево` - Предыдущий трек
- `Shift+Вправо` - Перемотка вперед на 10 секунд
- `Shift+Влево` - Перемотка назад на 10 секунд
- `F5` - Повторное сканирование папки
- `Ctrl+F` - Фокус на поисковой строке
- `Ctrl+M` - Включить/выключить звук
- `Ctrl+L` - Загрузить плейлист
- `Ctrl+S` - Сохранить плейлист
- `Esc` - Очистить поиск

### Перетаскивание:
- Перетащите папку для установки музыкальной директории
- Перетащите аудиофайлы для добавления в библиотеку

## Режимы воспроизведения

- **По порядку**: Воспроизведение треков последовательно
- **Случайный**: Воспроизведение треков в случайном порядке
- **Повтор**: Бесконечное повторение текущего трека
- **Повтор всего**: Повторение всего плейлиста

## Технические детали

- **Аудио движок**: Windows Media Player COM компонент
- **Метаданные**: TagLib для извлечения информации из аудиофайлов
- **UI фреймворк**: Windows Forms (.NET)
- **Настройки**: INI-файл конфигурации
- **Производительность**: Асинхронное сканирование файлов для больших библиотек

## Решение проблем

### Частые проблемы:

1. **Приложение не запускается**:
   - Установите [.NET Framework 4.7.2](https://dotnet.microsoft.com/download/dotnet-framework/net472)
   - Убедитесь, что все DLL файлы находятся в той же директории

2. **Нет звука**:
   - Проверьте системную громкость
   - Убедитесь в целостности аудиофайлов
   - Проверьте кнопку отключения звука в приложении

3. **Файлы не загружаются**:
   - Проверьте права доступа к папке
   - Убедитесь в поддержке форматов файлов
   - Убедитесь, что файлы не повреждены

4. **Метаданные не отображаются**:
   - Некоторые файлы могут не содержать встроенных метаданных
   - Приложение будет отображать имя файла в качестве замены
