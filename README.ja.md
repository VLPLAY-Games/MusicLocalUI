高度な機能と直感的なインターフェースを備えた軽量なWindows用ローカルオーディオプレイヤー。

## 特徴

- **コンパクトサイズ**: 合計約1MBのみ
- **対応フォーマット**: MP3, WAV, FLAC, AAC, OGG, WMA, M4A
- **高度な再生制御**:
  - 再生/一時停止/次/前
  - クリックシーク対応のプログレスバー
  - ミュート機能付き音量制御
  - 再生履歴ナビゲーション
- **複数の再生モード**:
  - 順序（シーケンシャル）
  - ランダム（シャッフル）
  - リピート（単曲）
  - 全曲リピート（全リスト）
- **スマートライブラリ管理**:
  - 進捗表示付きフォルダスキャン
  - リアルタイム検索とハイライト表示
  - 永続ストレージ対応のお気に入りシステム
  - プレイリスト保存/読み込み（M3U形式）
- **豊富なトラック情報**:
  - 再生時間表示（hh:mm:ss）
  - ビットレートとファイルサイズ
  - 完全なメタデータ（アーティスト、アルバム、ジャンル、年）
  - アルバムアート表示
  - ファイル作成日
- **ユーザーエクスペリエンス**:
  - ファイルとフォルダのドラッグ＆ドロップ対応
  - 主要機能のキーボードショートカット
  - 視覚的なお気に入り表示
  - カスタムリストボックスレンダリング
  - 設定の永続化

## システム要件

- **OS**: Windows 7以降
- **.NET Framework**: 4.7.2以降
- **ディスク容量**: 2MB未満
- **RAM**: 最小512MB

## インストール

1. GitHubから最新リリースをダウンロード
2. ZIPファイルを任意の場所に展開
3. `MusicLocalUI.exe`を実行

### 必要なファイル:
- `MusicLocalUI.exe`（メインアプリケーション）
- `TagLibSharp.dll`（メタデータライブラリ）

## 使用方法

### 基本的な再生:
1. 「Open」をクリックして音楽フォルダを選択
2. 「Scan」をクリックしてすべてのオーディオトラックを読み込み
3. 任意のトラックをダブルクリックで再生開始
4. 再生コントロールを使用: ⏸ ▶ ⏭ ⏮

### 高度な機能:
- **検索**: 検索ボックスに入力してトラックをリアルタイムフィルタリング
- **お気に入り**: 「♡ Favorite」をクリックしてお気に入りトラックをマーク
- **プレイリスト**: 「Save Playlist」/「Load Playlist」でセッション管理
- **履歴**: 戻る/進むボタンで再生履歴をナビゲート
- **表示モード**: 「Show All」と「Show Favorites」を切り替え

### キーボードショートカット:
- `スペース` - 再生/一時停止
- `Ctrl+右` - 次のトラック
- `Ctrl+左` - 前のトラック
- `Shift+右` - 10秒早送り
- `Shift+左` - 10秒巻き戻し
- `F5` - フォルダ再スキャン
- `Ctrl+F` - 検索ボックスにフォーカス
- `Ctrl+M` - ミュート切り替え
- `Ctrl+L` - プレイリスト読み込み
- `Ctrl+S` - プレイリスト保存
- `Esc` - 検索クリア

### ドラッグ＆ドロップ:
- フォルダをドロップして音楽ディレクトリを設定
- オーディオファイルをドロップしてライブラリに直接追加

## 再生モード

- **順序**: トラックを順番に再生
- **ランダム**: トラックをランダム順で再生
- **リピート**: 現在のトラックを無限にリピート
- **全曲リピート**: プレイリスト全体をリピート

## 技術詳細

- **オーディオエンジン**: Windows Media Player COMコンポーネント
- **メタデータ**: オーディオファイル情報抽出にTagLib使用
- **UIフレームワーク**: Windows Forms (.NET)
- **設定**: INIベースの設定ファイル
- **パフォーマンス**: 大規模ライブラリ用の非同期ファイルスキャン

## トラブルシューティング

### 一般的な問題:

1. **アプリケーションが起動しない**:
   - [.NET Framework 4.7.2](https://dotnet.microsoft.com/download/dotnet-framework/net472)をインストール
   - すべてのDLLファイルが同じディレクトリにあることを確認

2. **音が鳴らない**:
   - システム音量を確認
   - オーディオファイルの完全性を確認
   - アプリケーション内のミュートボタンを確認

3. **ファイルが読み込まれない**:
   - フォルダの権限を確認
   - 対応ファイル形式を確認
   - ファイルが破損していないことを確認

4. **メタデータが表示されない**:
   - 一部のファイルには埋め込みメタデータが含まれていない場合があります
   - アプリケーションは代替としてファイル名を表示します
